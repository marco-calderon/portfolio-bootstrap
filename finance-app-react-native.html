Finance Manager Mobile App

This application is an implementation of an external design in React Native. It combines web technologies to offer financial management to the user, inform of expenses and incomes, and shows this information in charts and tables.

It is available for running on iOS, Android, and web.

The technology stack used is based on Expo, a framework for using native libraries on React Native for easy deployment into the three platforms (iOS, Android and Web). It uses storage native drivers for persisting information on the device.

While it donâ€™t have a backend implementation for storing the data on the cloud, the data layer and service allows this to be ported easily.

Insights can be shown in a animated chart inside the application. They are belong to the incomes, outcomes and assets values, and are implemented Victory Charts Native. All the code is well organized to easy for plug-and-play tasks, avoid styles collision and still gives a good performance with lots of information to be displayed.

The code is splitted to have React best coding practices in 4 component types: simple, containers, data management and page components. These have been used with TypeScript to ensure data consistency between components and state of application.

              
  import React from 'react';
  import { View, Text, StyleSheet, TouchableOpacity } from 'react-native';
  import Transaction from '../../components/Transaction';
  import { TransactionModel } from '../../models/transaction.model';
  
  export interface TransactionContainerProps {
    transactions: TransactionModel[];
    onAddClick?: Function;
  }
  
  const TransactionsContainer = (props: TransactionContainerProps) => {
    const { transactions, onAddClick } = props;
    return (
      <View style={styles.container}>
        <View style={styles.headerContainer}>
          <Text style={styles.title}>Transactions</Text>
          <TouchableOpacity style={styles.addButton} onPress={() => onAddClick && onAddClick()}>
            <Text style={styles.addButtonText}>Add</Text>
          </TouchableOpacity>
        </View>
        <View style={styles.transactionContainer}>
          {transactions && transactions.length > 0 && transactions.map(t => <Transaction key={t.id} transaction={t} />)}
        </View>
      </View>
    );
  }
  
  const styles = StyleSheet.create({
      ...
  });
  
  export default TransactionsContainer;
